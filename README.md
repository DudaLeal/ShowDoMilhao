# Show do Milhão

O jogo foi desenvolvido em HTML, CSS e JavaScript. 
Ele contém a função mínima do jogo Show do Milhão, que é a pergunta e resposta. 
As perguntas estão em modo aleatório e o jogo termina na 16ª pergunta. Você pode jogar quantas vezes quiser. 

Vídeo tutorial: https://youtu.be/uK_mD0ltcIQ
